# Excel-Hospital-Emergency-Room-Dashboard
This project is an interactive Hospital Emergency Room Dashboard built in Microsoft Excel to analyze patient data and provide actionable insights into emergency room operations. The dashboard enables users to monitor key healthcare metrics through dynamic visualizations and filters.
